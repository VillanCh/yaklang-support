# yak README

yak 是 yaklang 的语言支持，可以帮助用户更快编写 yak 脚本

## Features

1. 支持功能库的自动补全和功能库自动提示

### 当前内部版本 Release Note

### 1.1.0
**支持代码格式化与代码调试(测试中)**
调试的一个示例的launch.json如下:
```json
{
	"version": "0.2.0",
	"configurations": [
		{
			"type": "yak",
			"request": "launch",
			"name": "Debug yaklang program",
			"program": "${file}",
		}
	]
}

```
